# customised-spark-hadoop-hive

A pair of projects, customised to support newer versions of spark, and to make it work with spark connect.

Also modified to have a persistent volume on hive so the catalogue persists.

Docker-Spark sourced from: https://github.com/big-data-europe/docker-spark

Docker-Hadoop-Hive sourced from: https://github.com/Marcel-Jan/docker-hadoop-spark
